## create_anonymous_token

Create an authentication token for an anonymous user of external search.

### Example Usage

```python
from glean import Glean
import os


with Glean(
    bearer_auth=os.getenv("GLEAN_BEARER_AUTH", ""),
) as g_client:

    res = g_client.client.authentication.create_anonymous_token()

    # Handle response
    print(res)

```

## create_token

Creates an authentication token for the authenticated user.

### Example Usage

```python
from glean import Glean
import os


with Glean(
    bearer_auth=os.getenv("GLEAN_BEARER_AUTH", ""),
) as g_client:

    res = g_client.client.authentication.create_token()

    # Handle response
    print(res)

```
